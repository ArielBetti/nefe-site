Núcleo de Estudos em Front-End

Grupo idealizado para o aprofundamento nas tecnologias HTML e CSS e o estudo das novas tecnologias Front-End.

Propósito do nosso grupo

Aprofundar em HTML e CSS

As tecnologias HTML e CSS são pilares indispensáveis no mundo web. Portanto, o objetivo inicial do NEFE seria o aprofundamento e a padronização de conhecimentos, de modo que fossem abordados conceitos na teoria e na prática.

Descobrir novas tecnologias

A web é uma área que está em constante mudança, a cada dia novas tecnologias surgem e outras se tornam ultrapassadas. Sendo assim, o estudo constante se torna um dos objetivos principais do NEFE, sempre buscando aprender estas novas tecnologias para o Front-End.
